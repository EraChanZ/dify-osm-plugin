# Privacy Policy for OpenStreetMap Plugin

This plugin provides integration with the OpenStreetMap Overpass API.

## Data Collection

- **Location Data**: This plugin processes latitude and longitude coordinates provided by the user or the AI agent to perform searches on OpenStreetMap.
- **No Storage**: The coordinates and search results are processed in real-time and are **not stored** by the plugin or the developer.
- **Third-Party Service**: Search queries are sent to the public Overpass API (`overpass-api.de`). Please refer to the [OpenStreetMap Privacy Policy](https://wiki.osmfoundation.org/wiki/Privacy_Policy) for information on how they handle data.

## Data Usage

The data is used solely to provide the requested search results within the Dify platform.

## Contact

For any questions regarding this privacy policy, please contact the plugin author.
